# YouTube Playlist Progress Tracker

A simple Chrome extension that helps you actually finish YouTube playlists — because once you see your progress, you naturally want to keep going.

---

## Why I built this

YouTube already shows that tiny red line under videos.

But let’s be honest…  
that doesn’t really help.

You still don’t know:
- how much you’ve completed  
- how much is left  
- how close you are to finishing  

And because of that, it’s easy to stop halfway.

When you can’t see your progress, you lose motivation.  
But when you can see it — you feel like finishing it.

---

## What this extension does

- Automatically detects when you open a playlist  
- Figures out which videos you’ve already watched  
- Shows a clean popup with:
  - number of videos completed  
  - total videos  
  - progress percentage  
- Adds a simple progress bar  
- Extracts topics from video titles so you know what you’re learning  
- Saves your progress even after closing the browser  
- Lets you reset anytime  

---

## The main idea

This whole extension is based on one simple thing:

**Seeing progress pushes you to work more.**

Instead of thinking:
> “Where did I stop?”

You start thinking:
> “I’ve already come this far… might as well finish it.”

That small shift makes a big difference.

---

## Why it’s useful

This is helpful if you:
- follow coding playlists  
- learn from YouTube courses  
- don’t like leaving things unfinished  

It makes your learning feel more structured and motivating.

---

## How it works (in simple terms)

- Looks at the playlist page  
- Reads the red progress bar YouTube already shows  
- Calculates your overall progress  
- Updates automatically while you browse  

---

## Experience

- Clean and minimal popup  
- Dark theme  
- Draggable interface  
- Non-intrusive design  

---

## How to use

1. Go to `chrome://extensions`  
2. Turn on **Developer Mode**  
3. Click **Load unpacked**  
4. Select the project folder  

Now open any playlist — it works automatically.
